Data Deletion Request – iDentity App

The iDentity app does not collect, store, or transmit any personal data on external servers. All user profiles and app data are stored locally on the device only.

How to delete your data

Users can delete all app data at any time by:

  1. Opening the app
  2. Going to the profile settings
  3. Selecting “Delete Profile”
  4. Confirming the deletion

This permanently removes all data stored on the device.
